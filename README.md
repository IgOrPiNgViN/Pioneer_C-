# Курс C++ - Дворец Пионеров

Этот репозиторий содержит материалы курса программирования на C++ для учащихся Дворца Пионеров.

## Описание курса

Курс предназначен для изучения основ программирования на языке C++ и включает в себя:

- 30 уроков с теоретическими материалами и практическими заданиями
- Примеры кода для каждого урока
- Упражнения для закрепления материала
- Домашние задания
- Проекты для практического применения знаний

## Как использовать

1. Каждый урок содержит теоретическую часть с объяснениями
2. В каждом файле есть примеры кода для практики
3. Выполняйте упражнения для закрепления материала
4. Решайте домашние задания
5. Создавайте собственные проекты на основе изученного

## Требования

- Компилятор C++ (например, Visual Studio, Code::Blocks, Dev-C++)
- Базовые знания математики
- Желание изучать программирование
