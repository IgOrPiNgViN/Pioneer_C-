**Вопрос 2 (5 баллов)**  
✅ `++x` — сначала увеличивает переменную на 1, затем возвращает новое значение.  
✅ `x++` — сначала возвращает текущее значение, затем увеличивает переменную на 1.  

Пример:
```cpp
int x = 5; int y = ++x;  // x = 6, y = 6
int x = 5; int y = x++;  // x = 6, y = 5
```

*Критерии:*
- Объяснение префиксного инкремента — 2 балла
- Объяснение постфиксного инкремента — 2 балла
- Корректный пример — 1 балл

✅ Оператор `*` — разыменование (получение значения по адресу).

*Критерии:*

**Вопрос 4 (5 баллов)**

✅ `do-while` проверяет условие ПОСЛЕ выполнения тела цикла (выполнится хотя бы один раз).  
✅ `do-while` используют, когда нужно выполнить код хотя бы один раз (например, меню, валидация ввода).

*Критерии:*
