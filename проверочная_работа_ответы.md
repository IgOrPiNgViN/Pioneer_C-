## ВАРИАНТ 1

### Теоретическая часть

**Вопрос 1** (Урок 1: Введение)  
- `cout` — оператор вывода данных на экран (консоль)
- `endl` — манипулятор, который переводит курсор на новую строку и очищает буфер
- `using namespace std;` — позволяет использовать имена из стандартной библиотеки без префикса `std::`

**Вопрос 2** (Урок 2: Переменные)  
- Переменная — это именованная область памяти для хранения данных определённого типа
- Правила именования:
  - Может содержать буквы, цифры, символ подчёркивания
  - Не может начинаться с цифры
  - Не может быть ключевым словом C++
  - Регистр имеет значение
- Правильно: `studentAge`, `my_var`, `count1`
- Неправильно: `2name`, `int`, `my-var`

**Вопрос 3** (Урок 3: Операторы)  
- `=` — оператор присваивания (присваивает значение переменной)
- `==` — оператор сравнения (проверяет равенство, возвращает bool)
- Примеры: `x = 10;` (присваивание), `if (x == 10)` (сравнение)

**Вопрос 4** (Урок 9: Структуры)  
- Структура — пользовательский тип данных, объединяющий переменные разных типов под одним именем
- Через переменную: `point.x`
- Через указатель: `ptr->x` или `(*ptr).x`

**Вопрос 5** (Урок 10: Функции)  
- Функция — именованный блок кода, выполняющий определённую задачу
- Объявление (прототип) — сообщает компилятору о существовании функции (тип, имя, параметры)
- Определение — содержит фактический код функции

### Практическая часть

**Задание 1:** `21`
```
a = 15, b = 4
result = 15 % 4 + 15 / 4 = 3 + 3 = 6
result++ → result = 7
result *= 3 → result = 21
```

**Задание 2:** `6`
```
Числа от 1 до 10, которые делятся на 2 И на 3 (т.е. на 6): только 6
sum = 6
```

**Задание 3:** `23`
```
p = {5, 10}
ptr->x = 5 * 2 = 10 → p.x = 10
ptr->y = p.x + 3 = 10 + 3 = 13 → p.y = 13

```

## ВАРИАНТ 2

### Теоретическая часть

**Вопрос 1** (Урок 2: Переменные)  
- `short` — короткое целое (обычно 2 байта)
- `int` — целое число (обычно 4 байта)
- `long` — длинное целое (4 или 8 байт)
- `long long` — очень длинное целое (8 байт)
- Отличаются размером в памяти и диапазоном значений

**Вопрос 2** (Урок 3: Операторы)  
- `%` — возвращает остаток от деления
- Пример: `10 % 3 = 1`
- Проверка чётности: `if (число % 2 == 0)` — число чётное

**Вопрос 3** (Урок 5: Циклы)  
- `for` — цикл со счётчиком, когда известно количество итераций
- `while` — цикл с условием, когда количество итераций неизвестно
- `for` удобнее для перебора массивов, `while` — для ввода до условия

**Вопрос 4** (Урок 6: Массивы)  
- Объявление: `int arr[5];` или `int arr[5] = {1, 2, 3, 4, 5};`
- Нумерация начинается с 0
- Третий элемент: `arr[2]` (индекс = позиция - 1)

**Вопрос 5** (Урок 10: Функции)  
- По значению — создаётся копия, изменения не влияют на оригинал
- По ссылке (`&`) — функция работает с оригиналом, изменения сохраняются

## ВАРИАНТ 3

### Теоретическая часть

**Вопрос 1** (Урок 2: Переменные)  
- Константа — переменная, значение которой нельзя изменить после инициализации
- Объявление: `const int MAX = 100;`
- Изменить значение константы нельзя — это вызовет ошибку компиляции

**Вопрос 2** (Урок 3: Операторы)  
- `&&` (И) — true, если ОБА операнда true
- `||` (ИЛИ) — true, если ХОТЯ БЫ ОДИН операнд true
- Примеры: `(5 > 3) && (2 < 4)` = true, `(5 > 3) || (2 > 4)` = true

**Вопрос 3** (Урок 4: Условия)  
- Без `break` произойдёт "проваливание" — выполнятся все последующие case
- `switch` работает с целыми типами: `int`, `char`, `enum`
- НЕ работает с `float`, `double`, `string`

**Вопрос 4** (Урок 7: Строки)  
- C-строки (`char[]`) — массив символов с '\0' в конце, фиксированный размер
- `string` — класс с динамическим размером, автоматическое управление памятью
- Длина string: `str.length()` или `str.size()`

**Вопрос 5** (Урок 10: Функции)  
- Рекурсивная функция — функция, которая вызывает сама себя
- Базовый случай — условие выхода из рекурсии, без него будет бесконечная рекурсия

**Задание 3:** `28`

## ВАРИАНТ 4

### Теоретическая часть

**Вопрос 1** (Урок 2: Переменные)  
- `bool` — логический тип данных
- Может принимать только два значения: `true` или `false`
- `cout` выводит: `true` как `1`, `false` как `0`

**Вопрос 2** (Урок 3: Операторы)  
- `++x` (префиксный) — сначала увеличивает, потом использует
- `x++` (постфиксный) — сначала использует, потом увеличивает
- Пример: `int x = 5; int y = ++x;` → x=6, y=6
- Пример: `int x = 5; int y = x++;` → x=6, y=5

**Вопрос 3** (Урок 5: Циклы)  
- `break` — полностью прерывает выполнение цикла
- `continue` — пропускает оставшуюся часть текущей итерации и переходит к следующей

**Вопрос 4** (Урок 8: Указатели)  
- Указатель — переменная, хранящая адрес другой переменной
- `&` — получение адреса переменной
- `*` — разыменование (получение значения по адресу)

**Вопрос 5** (Урок 10: Функции)  
- Параметры по умолчанию — значения, которые используются, если аргумент не передан
- Пример: `void greet(string name = "Гость");`
- Вызов: `greet();` использует "Гость", `greet("Иван");` использует "Иван"
