# Инструкция по использованию русского языка в C++

## Проблема

В Windows консоли русские символы часто отображаются как вопросительные знаки или кракозябры из-за неправильной кодировки.

## Решение

### Способ 1: Использование заголовочного файла (Рекомендуется)

1. **Добавьте файл `russian_support.h`** в папку с вашими исходными файлами.

2. **Подключите заголовочный файл** в начале вашей программы:

   ```cpp
   #include "russian_support.h"
   ```

3. **Вызовите функцию `setupRussian()`** в самом начале функции `main()`:

   ```cpp
   int main() {
       setupRussian();  // ВАЖНО: вызвать в начале!
       
       // Ваш код здесь
       cout << "Привет, мир!" << endl;
       
       return 0;
   }
   ```

### Способ 2: Прямое добавление кода (Без отдельного файла)

Если вы не хотите создавать отдельный файл, добавьте этот код в начало вашей программы:

```cpp
#include <iostream>
#ifdef _WIN32
    #include <windows.h>
#endif
#include <locale>

using namespace std;

int main() {
    // Настройка русского языка
    #ifdef _WIN32
        SetConsoleOutputCP(65001);
        SetConsoleCP(65001);
        setlocale(LC_ALL, "Russian");
    #else
        setlocale(LC_ALL, "ru_RU.UTF-8");
    #endif
    
    // Ваш код здесь
    cout << "Привет, мир!" << endl;
    
    return 0;
}
```

## Важные замечания

1. **Вызывайте `setupRussian()` в самом начале `main()`** - до любого вывода на консоль.

2. **Кодировка файлов**: Убедитесь, что ваши `.cpp` файлы сохранены в кодировке **UTF-8**.

3. **Visual Studio**: Если используете Visual Studio:
   - Перейдите: `File -> Advanced Save Options`
   - Выберите: `Unicode (UTF-8 with signature) - Codepage 65001`
   - Или добавьте флаг компилятора: `/utf-8`

## Пример использования

Смотрите файл `пример_использования_русского.cpp` для полного примера.

## Как это работает

Функция `setupRussian()` делает следующее:

- Устанавливает кодировку консоли Windows в UTF-8 (код 65001)
- Настраивает локаль для русского языка
- Обеспечивает корректный ввод и вывод русских символов

## Результат

После применения этого решения, все русские символы будут отображаться корректно:

```
Введите ваше имя: Иван
Здравствуйте, Иван!
```
