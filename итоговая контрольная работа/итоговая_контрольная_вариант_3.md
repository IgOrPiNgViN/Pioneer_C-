**Задание 5 (14 баллов)** — *Урок 16. Перегрузка операторов*  
В классе `Vector` перегружен `operator+`, но в коде **две ошибки**. Найдите их и напишите исправленный вариант.

```cpp
class Vector {
    int x, y;
public:
    Vector(int a, int b) : x(a), y(b) {}
    Vector operator+(Vector other) {
        return Vector(x + other.x, y + other.y);
    }
    void print() { cout << x << " " << y; }
};

int main() {
    Vector v1(1, 2);
    Vector v2(3, 4);
    Vector v3 = v1 + v2;
    v3.print();
    Vector v4 = v1 + 5;   // должно ли компилироваться?

}
```
